## Translation Rules

1. Only translate VALUES (right side of `=` and after `.desc =`). Never change keys like `ent-SomethingHere`.
2. Preserve ALL Fluent syntax: `{$variables}`, `{CAPITALIZE($name)}`, `{INDEFINITE($entity)}`, `.desc =` attributes, selectors, multiline blocks.
3. Keep proper nouns, chemical formulas, brand names, game-specific terms (Syndicate, Nanotrasen, PDA) in English.
4. Use natural Spanish - informal "tu" form, standard Latin-neutral Spanish.
5. Common translations: station=estacion, crew=tripulacion, shuttle=lanzadera, engineering=ingenieria, medical=medico/a, security=seguridad, captain=capitan, cargo=carga, science=ciencia, power=energia, atmos=atmosfericos, airlock=esclusa, emitter=emisor, singularity=singularidad.
6. If a value contains ONLY Fluent references like `{ ent-SomeOtherEntity }` or `{ "" }`, do NOT translate - keep as-is.
